# 香港中小學電腦派位模擬器

> 整合教育局 2024/2026 官方數據的 SSPA 升中派位 + POA 小一入學模擬系統

🔗 **Live Demo:** https://altezza938.github.io/sspa/

## 功能

- **SSPA（中學學位分配辦法）** — 甲部 3 個不限區志願 + 乙部 30 個校網志願，依教育局機制以大 Band (1/2/3) 決定分配優先次序
- **POA（小一入學統籌辦法）** — 計分制（最高 35 分）+ 統一派位（甲部約10% + 乙部約40%）
- 全港 400+ 中學真實學校數據庫（教育局編號、非保留學額、聯繫學額）
- AI 智能推薦志願

## EDB 機制說明

### SSPA
- 學生按呈分試成績分為全港 Band 1/2/3（大班）
- 同一大 Band 內，以電腦隨機編號決定優先次序（號碼越大越優先）
- 甲部（不限地區）先處理，再處理乙部（按校網）
- 學校學額按 Band 1 → 2 → 3 順序填滿；若 Band 1 學額用盡仍有剩餘，Band 2/3 學生亦可獲派

### POA
- 自行分配學位：必收生（世襲生）保證錄取；計分制最高 35 分（第1-5類選一最高20分 + 第6-7類選一最高5分 + 年齡10分）
- 統一派位：甲部（不限地區，約10%）+ 乙部（按校網，約40%）
